Функция Zip нужня для объединения списков. Объединения там хитрое.

```python
l1 = [1, 2, 3]
l2 = [4, 5, 6]
k = list(zip(l1, l2))
print(k)


>>>  [(1, 4), (2, 5), (3, 6)]
```

Мы получаем список из кортежей. Важно, чтобы длины всех списков были одинаковыми.
